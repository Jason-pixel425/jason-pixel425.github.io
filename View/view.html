Random Fortune - JavaScript

This is a portfolio project that was created as part of the Codecademy Full-stack engineering career course. This simple program logs a somewhat random fortune to the console. There are four words that are randomly selected from four seperate random arrays. The program was constructed this way instead of being completly random so that the sentence produced makes sense

It can also be expanded upon in the future via the following:

  • Adding different sentence foundations for the words to be inserted in. These foundations would be randomly selected.
  • Adding more words to be selected in the arrays
  • Making more of the words random while ensuring the sentence still makes sense
